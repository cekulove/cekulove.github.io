首先，声明一个指向常量的指针p

{% highlight cpp %}
int age = 29;
const int * p = &age;
{% endhighlight %}

{% highlight cpp %}
*p = 30;   //invalid
cin >> *p; //invalid
{% endhighlight %}

可以通过age变量来修改age的值,但*p的值为const,不允许修改



{% highlight cpp %}
*p = 20;   //invalid
age = 20;  //valid
{% endhighlight %}
